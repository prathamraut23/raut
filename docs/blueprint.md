# **App Name**: Raut Orange Ecosystem

## Core Features:

- Homepage Showcase: Display key information including company overview, vision, and objectives with image-rich sections and clear 'Learn More' calls to action.
- About Us Section: Detail the history, mission, and core values of Raut Farmer Producer Company with compelling visuals.
- Services Overview: Present all services like orange farming, sorting, and grading with dedicated pages and supporting imagery.
- Project Gallery: Showcase completed and ongoing projects related to agriculture and orange production with a visual gallery.
- Contact Form: Include a contact form with fields for name, mobile number, email, and message, alongside company contact details.
- Mobile Responsiveness: Ensure the website is fully responsive and optimized for viewing on various mobile devices.

## Style Guidelines:

- Primary color: Saturated orange (#F2711C), reminiscent of ripe oranges and the company's focus.
- Background color: Very light, desaturated orange (#F8EFE8) provides a warm, inviting backdrop.
- Accent color: Yellow-orange (#F29F05), offers a sunny, optimistic contrast to highlight key elements.
- Headline font: 'Belleza' (sans-serif) will create stylish, visually engaging titles.
- Body font: 'Alegreya' (serif) provides excellent readability in long blocks of text.
- Use modern, outline-style icons related to agriculture, farming, and orange production to enhance visual communication.
- Implement a clean, grid-based layout with generous white space and clearly defined sections for easy navigation.
